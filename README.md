# 🚀 Data Master Pro v3.0

An advanced, Python-based terminal tool designed for professional data entry and automated billing management. 

## ✨ Key Features
- 🔐 **Custom Security:** Set your own password with SHA-256 encryption.
- 🧾 **Auto-Invoicing:** Calculate gross total, discounts, and net payable automatically.
- 🔍 **Global Search:** Find any specific entry within seconds.
- 📊 **CSV Integration:** Data is stored in structured CSV files for easy export to Excel.
- 🎨 **Pro UI:** Stylish ASCII banners and color-coded error/success messages.
- 🕒 **Auto-Log:** Keeps track of entry times automatically.

## 🛠️ Requirements
- Python 3.x
- Termux or any Terminal Emulator
